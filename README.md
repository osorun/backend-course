## 🐰 Bunny Interactive Display

**Tech Stack:** JavaScript, HTML, CSS

**What I Built:** An interactive webpage that displays a JavaScript object with styling.

**Approach:** 
- Declared a `bunny` object with `name`, `age`, and `isHappy` properties
- Built an HTML card layout with a pink color scheme
- Used CSS for rounded corners, shadows, and responsive centering
- Connected JavaScript to HTML using `document.getElementById()` and template literals

**Challenges Faced:**
- Learned how to separate concerns: keeping JavaScript in `bunny.js` and linking it to HTML
- Figured out browser compatibility and Live Server usage

**How to Run:**
1. Open `index.html` in any browser (or use VS Code Live Server)

**Key Features:**
- 🐰 Pink-themed responsive card design
- 📋 Dynamic rendering of JavaScript object properties
- 🎨 Clean separation of HTML, CSS, and JavaScript
